## Descripción General

Este documento presenta la **propuesta de arquitectura** del Sistema de Control de Asistencia con reconocimiento facial, diseñado para operar exclusivamente en una **red local**, con una **base de datos centralizada** y procesamiento biométrico distribuido en cada equipo. La solución considera un entorno de menos de 10 equipos, con posibilidad de escalar a múltiples sedes, manteniendo altos niveles de seguridad y rendimiento.

Se analizan y comparan dos alternativas arquitectónicas: **arquitectura web en red local (intranet)** y **arquitectura de aplicación local cliente–servidor**. Cada alternativa es evaluada desde un enfoque técnico, funcional y operativo, considerando aspectos como control del hardware, seguridad, mantenimiento, escalabilidad y facilidad de despliegue. El objetivo es identificar la arquitectura más adecuada para el contexto actual del proyecto y establecer una base sólida para su evolución futura.
