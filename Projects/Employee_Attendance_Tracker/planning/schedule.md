# Cronograma del Proyecto

Este documento presenta la planificación inicial del proyecto, un sistema web local de control de asistencia orientado a pequeñas y medianas empresas (MYPES). El cronograma define las actividades principales del desarrollo, sus dependencias, los tiempos estimados y la asignación de responsabilidades, sirviendo como guía para la correcta ejecución del proyecto.

El cronograma representa una versión inicial validada y será utilizado como referencia para comparar la planificación con el avance real del proyecto al finalizar la implementación del prototipo.

La herramienta utilizada para su elaboración es **ProjectLibre**, permitiendo una gestión estructurada y visual del proceso de desarrollo.

## Resultados Esperados

Se espera que el cronograma facilite el seguimiento del proyecto Web, asegure el cumplimiento de los plazos establecidos y contribuya a una organización eficiente del trabajo durante el desarrollo del sistema.
