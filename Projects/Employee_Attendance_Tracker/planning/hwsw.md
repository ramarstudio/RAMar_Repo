## Recursos técnicos

Recursos técnicos que se utilizarán para el desarrollo del proyecto. Se especifican las características técnicas mínimas del hardware y del software de soporte asignado a las actividades de desarrollo.

### Hardware
- Equipo de cómputo con **mínimo 8 GB de memoria RAM**.
- **Procesador Intel Core i5 de décima generación** o superior.
- **Almacenamiento SSD de al menos 256 GB**.
- **Conexión a Internet estable**, con un bitrate mínimo de **10 Mbps de descarga** y **5 Mbps de subida**.

### Software
El software se agrupa según las herramientas necesarias para el desarrollo del proyecto.

- **Sistema Operativo:** Windows 11.
- **Lenguajes de desarrollo:** HTML5, CSS3 y JavaScript ES15.
- **Entorno de Desarrollo Integrado (IDE):** Visual Studio Code.
- **Sistema de Control de Versiones (VCS):** Git y GitHub.
