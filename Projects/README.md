# 📚 Centro de Especificaciones y Casos de Uso (Área de Planning)

Bienvenido al directorio de **Planificación e Investigación** de RAMar Software Studio. Toda la magia del código inicia aquí.

Este directorio no contiene código fuente ejecutable, sino que almacena toda la estructuración teórica, análisis de flujos, ingeniería de requerimientos (Matriz MoSCoW) y diagramas en los que se basan nuestros sistemas antes de pasar a la etapa de desarrollo.

A continuación, se listan los documentos clave que definen cómo, por qué y bajo qué contexto empresarial se construyó el sistema biométrico.

El documento formal de análisis de ingeniería de software. Determina tajantemente lo que el producto *"Debe Hacer" (Must)*, *"Debería Hacer" (Should)* y lo que queda fuera del alcance para garantizar el MVP.
