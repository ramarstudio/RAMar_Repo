<span class="section-label">Diferenciadores</span>

## Por que elegir este sistema

!!! success "Privacidad absoluta"
    El sistema **nunca almacena fotografias**. Convierte cada rostro en un vector matematico cifrado con AES-256. Si un atacante obtiene los datos, no puede reconstruir ningun rostro — el proceso es matematicamente irreversible.

!!! success "Sin internet, sin suscripciones"
    Todo opera dentro de la red local de la empresa. No hay costos recurrentes, no se envian datos a terceros, no hay puntos de fallo externos.

!!! success "Rendimiento nativo"
    Al ser una aplicacion de escritorio Windows (no web), accede directamente al hardware de la camara con un consumo de CPU menor al 1%. Sin las capas de abstraccion del navegador.
