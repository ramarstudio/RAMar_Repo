# Instalación

Esta sección cubre todo lo necesario para desplegar el sistema de asistencia biométrico en una máquina Windows.

## Tiempo estimado

| Paso | Tiempo |
|---|---|
| Instalar software (`.NET`, `Python`, `PostgreSQL`) | ~15 min |
| Configurar base de datos | ~2 min |
| Configurar `appsettings.json` | ~2 min |
| Instalar dependencias Python | ~5 min |
| Compilar y ejecutar | ~2 min |
| **Total** | **~26 min** |

!!! note "Primera ejecución"
    En la primera ejecución, el motor facial descarga los modelos de IA (~600 MB). Esto agrega ~5 minutos adicionales dependiendo de la velocidad de internet. Solo ocurre una vez.
