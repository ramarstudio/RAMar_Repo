## 🗂️ Portafolio de Proyectos Activos

### 1. [Attendance System (Control de Asistencia Biométrico)](./AttendanceSystem/README.md)
Un sistema nativo de clase corporativa que automatiza el control de asistencia de empleados bajo una eficiente arquitectura de **Microservicios Inter-procesos**. Integra interfaces limpias en `.NET WPF` para control de hardware y algoritmos avanzados de **Reconocimiento Facial mediante IA**. Maximiza la privacidad biométrica operando en entornos 100% *Offline* (Sin requerir salida a la nube externa).
